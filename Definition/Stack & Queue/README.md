# Stack and Queue(co-authorized with gpt)

## Stack

### Definition
A stack is a linear data structure that follows the
Last In First Out (LIFO) principle.

### Common Operations
- Push
- Pop
- Peek

### Time Complexity
- Push: O(1)
- Pop: O(1)
- Peek: O(1)

### Applications
- Function calls
- Expression evaluation
- Undo / Redo

---

## Queue

### Definition
A queue is a linear data structure that follows the
First In First Out (FIFO) principle.

### Types
- Simple Queue
- Circular Queue
- Deque
- Priority Queue

### Time Complexity
- Enqueue: O(1)
- Dequeue: O(1)

### Applications
- Scheduling
- Buffer management
- BFS traversal

## Common Mistakes
- Stack overflow / underflow
- Forgetting circular condition in queue
